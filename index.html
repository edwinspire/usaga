Welcome to OpenSAGA Page.

OpenSAGA is a project to build an alarm system administrator (SAGA stands for automatic alarm management in Spanish).

It was initially oriented toward community alarms with SMS notifications, HOWEVER currently the project continues to grow.

Originally written in C# and MySQL as database currently is moving all code to VALA and PostgreSQL.

It has a Web interface, an embedded http server.

The project requires collaboration, any help is welcome and will be economically, reporting bugs or collaborating with ideas for improvement.

OpenSAGA requires the following libraries:
